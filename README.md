Instruções:
1 - Baixe os arquivos
2 - Instale o venv
3 - instale as dependencias 
    bash: python -m pip install -r requirements.txt
4 - gere a base de dados 
    bash: python gerar_dados.py
5 - treine o modelo
    bash: python treinar_modelo.py
6 - rode a api
    bash: python api.py
    acesse: http://127.0.0.1:5000
